Опис проєкту

«Шлях Блюзмена» — ігровий програмний застосунок у жанрі тактичної карткової RPG, створений на рушії Unity з використанням C#. У грі реалізовано головне меню, карту світу, маршрут регіону, карткову бойову систему, нагороди, артефакти, магазин, події, табір, інвентар, кодекс, налаштування, збереження прогресу та фінальний екран завершення проходження.
